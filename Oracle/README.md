## Oracle Analytics 練習
Oracle Analytics 這部分老師並未授課教學使用，僅提供資料讓學員自行操作。這次拿到的資料是某間公司的員工資料，並無業績、考核...這類可做KPI的資訊，比較偏向員工個人基本資料，
所以這次製作的方向會是：提供給人資部門的員工基本資料分析。<br>
